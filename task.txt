Framework: FastAPI
Database: PostgreSQL
ORM: SQLAlchemy (async) + Alembic for migrations
Auth: JWT using python-jose + passlib bcrypt
Scheduler: APScheduler for fetching news every 30 mins from Currents API
WebSocket: FastAPI native WebSockets with a ConnectionManager class
Push Notifications: Firebase FCM via firebase-admin SDK
News Source: Currents API
Cache: Redis
Background Tasks: Celery + Redis
